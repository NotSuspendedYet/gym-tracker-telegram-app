# Проверка настройки Telegram бота

## 1. Откройте @BotFather в Telegram

## 2. Отправьте команду:
```
/mybots
```

## 3. Выберите вашего бота

## 4. Проверьте Bot Settings → Menu Button

**ДОЛЖНО БЫТЬ:**
```
URL: https://gymtrackerbot.ru
```

---

## Важно!

❌ **НЕ ОТКРЫВАЙТЕ** приложение напрямую в браузере: https://gymtrackerbot.ru

✅ **ОТКРЫВАЙТЕ ТОЛЬКО** через:
- Кнопку меню бота в Telegram (☰ рядом с полем ввода)
- Или через команду /start в боте

---

## Если Menu Button не настроена:

1. @BotFather → /mybots → ваш бот
2. Bot Settings → Menu Button
3. Set Menu Button → Enter URL: `https://gymtrackerbot.ru`
4. Done!

---

## После настройки:

1. Откройте вашего бота в Telegram
2. Нажмите кнопку меню (☰) или /start
3. Приложение откроется ВНУТРИ Telegram
4. Теперь вы увидите своё имя и все данные сохранятся!
